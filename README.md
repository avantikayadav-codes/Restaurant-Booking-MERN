# 🍽️ Restaurant Booking - MERN Stack Web App

A full-stack restaurant booking web application built using the MERN stack, 
allowing users to explore the menu, learn about the restaurant, and book a table online.

---

## ✨ Features

- 🏠 **Hero Section** — Beautiful landing page
- 🍴 **Menu Display** — Browse restaurant menu
- 📅 **Table Reservation** — Book a table online
- 👥 **Team Section** — Meet the team
- ℹ️ **About Us** — Restaurant story
- ⭐ **Customer Reviews** — User testimonials
- 📱 **Responsive Design** — Works on all devices
- ❌ **404 Page** — Custom not found page

## ⚙️ How to Run

### Backend
```bash
cd Backend
npm install
node server.js
```

### Frontend
```bash
cd Frontend
npm install
npm run dev
```
